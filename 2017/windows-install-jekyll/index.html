1. 安装 Ruby

官网：http://rubyinstaller.org/downloads/

测试： Ruby是否安装成功，执行命令ruby -v

注意：勾选 “Add Ruby executables to your PATH”，安装路径不能包含空格

b. 安装 Easy Install

注意：添加 Python Scripts 路径到环境变量
